# 그래프
![2742E4435624E03F0C](https://user-images.githubusercontent.com/37062337/101359854-2afd3880-38e0-11eb-88b7-1403982d8cdf.png)
- 정점과 간선으로 이루어진 자료구조다. 정확히는 정점간의 관계를 표현하는 조직도라고 할 수 있으며, 트리가 대표적인 그래프 형태라고 할 수 있다.
- 방향이 있을수도, 없을수도 있다.
- 그래프를 탐색 : 하나의 정점으로부터 시작해 차례대로 모든 정점들을 한 번씩 방문하는 것이다.
- DFS, BFS, 다익스트라, 플로이드 와샬 등등 다양한 그래프 탐색 방법이 있다.

### 인접 행렬과 인접 리스트
- 인접 행렬 : 그래프의 노드를 2차원 배열형태로 만들어서 표현한다.
